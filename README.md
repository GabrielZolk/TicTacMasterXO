# TicTacMasterXO

A feature-rich Tic-Tac-Toe game for Android — much more than the classic 3×3.

**Live on Google Play** ⭐ 4.8

## Features
- 🌐 Online multiplayer
- 🤖 AI opponent with multiple difficulty levels
- ⚡ Special game modes (Blitz, Gravity, Big Board)
- 🎨 Themes & customization
- 📊 Statistics tracking

## Tech stack
React Native (Expo) · TypeScript · Firebase · AdMob
